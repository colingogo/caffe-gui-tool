Installation

Placeholder video while tutorial uploads

  • Download the x64 Linux blender build
  • Download or clone the two python scripts from the master branch
  • Run the blender application
  • File --> user preferences --> addons --> install from file --> select Caffe Nodes script --> install
  • Tick the checkbox in the newly appeared box's top right corner
  • File --> user preferences --> addons --> install from file --> select Caffe Generate script --> install
  • Tick the checkbox in the newly appeared box's top right corner
  • User preferences --> file --> uncheck relative paths (IMPORTANT)
  • Click on the cube icon in the bottom left
  • Click 'node editor' in the popup menu
  • In the newly appeared node editor, look at the bottom panel, it should read 'view - select - add - node'
  • Immediately right of this panel there should be a set of four icons joined together. select the THIRD.
  • File --> save startup file
  • File --> user preferences --> save user settings (on the bottom of the window(
  • Restart blender

Use

  • Click the 'New' button on the bottom part of the screen to create a new network
  • Name your network in this box, and ENSURE THE 'F' BUTTON NEXT TO THE NAME IS CHECKED
  • Use shift+a to start adding nodes
  • Join up, and fill in all the required fields to your taste. All networks require a data node, and solver node, with paths to data correctly entered.
  • The data node is fairly self explanatory, however in the data node, snapshot data is the path trained networks are saved, and config data is the path where prototxts are saved
  • When ready to train, select all nodes by single, (or double if required) tapping 'a', then press the spacebar.
  • Type 'solution' in the search bar that appears
  • Select 'create solution' and press enter; this will create the required prototxt files in the config directory.
  • Mark the generated train file in your config data path as executable
  • Run the train file
  • Smile
